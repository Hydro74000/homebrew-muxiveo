# homebrew-muxiveo

Homebrew tap for [Muxiveo](https://github.com/Hydro74000/muxiveo) — GUI video workflow tool for remuxing, encoding, Dolby Vision and HDR10+.

## Installation

```bash
brew tap Hydro74000/muxiveo
brew install muxiveo
```

## Migration depuis homebrew-mediarecode

Le projet **Mediarecode** a été renommé en **Muxiveo**. Pour migrer :

```bash
brew uninstall mediarecode
brew untap Hydro74000/mediarecode
brew tap Hydro74000/muxiveo
brew install muxiveo
```

## Mise à jour

```bash
brew update
brew upgrade muxiveo
```
